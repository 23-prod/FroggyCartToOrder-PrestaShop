1. Installation du module

1.1 Installation sur PrestaShop 1.4

Tout d'abord, connectez-vous sur votre panneau d'administration PrestaShop.

Une fois connecté, allez dans la section “Modules”.

Vous devriez avoir reçu un fichier nommé “froggycartorder.zip”. Vous allez devoir le télécharger dans cette section. Pour y parvenir, cliquez sur le bouton “Ajouter un module depuis mon ordinateur”.

Une nouvelle section devrait être apparu, cliquez sur le bouton “parcourir”.

Puis sélectionnez le fichier “froggycartorder.zip” et cliquez sur “ouvrir”.

Enfin, cliquez sur le bouton “Télécharger ce module”.

Un message de confirmation de téléchargement devrait apparaître.

Cliquez sur “Fonctionnalités du Front Office” et cherchez le module “Froggy Cart To Order” ou utilisez le filtre de recherche en tapant à l'intérieur “Froggy Cart To Order”, puis cliquez sur le bouton “Installer”.

Une fois que vous aurez cliqué sur le bouton d'installation, un message de confirmation “Module installé avec succès” devrait apparaître. Vous pouvez à présent cliquer sur le bouton “Configurer”.

1.2 Installation sur PrestaShop 1.5

Tout d'abord, connectez-vous sur votre panneau d'administration PrestaShop.

Une fois connecté, allez dans la section “Modules”.

Vous devriez avoir reçu un fichier nommé “froggycartorder.zip”. Vous allez devoir le télécharger dans cette section. Pour y parvenir, cliquez sur le bouton “Ajouter un module depuis mon ordinateur”.

Une nouvelle section devrait être apparu, cliquez sur le bouton “parcourir”.

Puis sélectionnez le fichier “froggycartorder.zip” et cliquez sur “ouvrir”.

Enfin, cliquez sur le bouton “Télécharger ce module”.

Votre navigateur devrait défiler automatiquement jusqu'au nouveau module ajouté, et vous devriez le voir en haut de votre écran. Maintenant vous pouvez cliquer sur le bouton “Installer”.

Une fois que vous aurez clique sur le bouton d'installation, un message de confirmation “Module installé avec succès” devrait apparaître. Vous pouvez à présent cliquer sur le bouton “Configurer” (sur certaines versions de PrestaShop, vous serez redirigé automatiquement vers la page de configuration après installation du module).

2. Utilisation du module

Voici ce que vous devriez avoir sur la page de configuration du module.

Pour convertir un panier en commande, vous pouvez soit choisir parmis les 50 derniers paniers créés (qui sont associés à un client mais pas encore à une commande).

Ou, si vous connaissez l'e-mail du client, commencez à renseigner l'e-mail (dans notre cas sheldon@cooper.com), les paniers associés au client (et non convertis en commande) seront affichés.

Une fois que vous aurez cliqué sur un panier (ou utilisé le menu déroulant), l'étape 2 apparaîtra.

Vous devez à présent choisir la méthode de paiement.
Vous pouvez soit choisir parmis les méthodes de paiement existantes (la liste est remplie avec toutes les méthodes de paiement qui ont déjà été utilisé).

Ou écrire un nouveau nom de méthode de paiement. Une fois que vous aurez sélectionné votre méthode de paiement (ou créé une nouvelle), l'étape 3 apparaîtra.
Attention, le nom de la méthode de paiement est utilisé uniquement pour l'affichage, aucun paiement ne sera effectué automatiquement, vous devrez vous charger de cette partie directement avec le client.

Choississez le statut de la commande qui sera assigné à cette nouvelle commande, alors l'étape 4 apparaîtra.

Enfin, cliquez sur le bouton "convertir".

Si tout s'est bien déroulé, un message de confirmation devrait apparaître.
